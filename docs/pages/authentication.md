Use `get_twitterapi_guide` for the offline snapshot (by page key), or `get_twitterapi_url` for URL/path-based fetches.

```json
{
  "tool": "get_twitterapi_guide",
  "arguments": {
    "guide_name": "authentication"
  }
}
```

```json
{
  "tool": "get_twitterapi_url",
  "arguments": {
    "url": "/authentication"
  }
}
```
